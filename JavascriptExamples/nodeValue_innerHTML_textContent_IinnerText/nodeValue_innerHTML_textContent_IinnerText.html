  • nodeValue: is a little more confusing to use, but faster than innerHTML.Thing is deprecated. No example, could not understand
  • innerHTML: parses content as HTML and takes longer. Example - innerHtml: Hello <em>cruel</em> world
  • textContent: uses straight text, does not parse HTML, and is faster. Example - textContent: Hello cruel world
  • innerText: Non-standard so should not be used. It generally is the equilvalent of what a user would get if they highlighted and copied some textr from a web page.Takes styles into page. It won't get hidden text for instance. Example innerText: Hello cruel world.
